Project Overview

Karnataka Heritage Explorer is a web-based travel and tourism guide application built using Python and Streamlit.
It helps users discover and explore the rich cultural, historical, spiritual, and natural heritage of Karnataka across all districts.

The app provides structured information about tourist places along with categories, descriptions, best visiting months, and direct Google Maps navigation links.

🚀 Features
📅 Explore places based on months (season-wise travel planning)
🏞 Browse destinations district-wise (A–Z sorted)
🔎 Smart search system (place / category / district)
🗺 One-click Google Maps navigation
👤 User Login & Sign Up system
📊 Structured dataset of Karnataka tourism places
🎯 Clean and simple Streamlit UI
🧠 Technologies Used
Python 🐍
Streamlit 🌐
SQLite3 🗄
Hashlib 🔐 (for password security)
Google Maps API (URL-based navigation)
